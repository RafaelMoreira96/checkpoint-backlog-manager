Importar CSV

Inserir lista de jogos

Insira um arquivo CSV com os gêneros que deseja adicionar ao banco de dados.
O arquivo deve conter uma coluna com o nome do gênero apenas.

Inserir backlog

Insira um arquivo CSV com os fabricantes que deseja adicionar ao banco de dados.
O arquivo deve conter uma coluna com o nome do fabricante e ano de fundação.

Instruções para manipular a importação de CSV

Caro administrador, para importar corretamente as listas no formato .csv, certifique-se que as informações apresentadas em cada aba sejam seguida a risca. Outra observação é: ao inserir uma lista de consoles, certifique-se de que as fabricantes estejam salvas no sistema, pois se não estiver salva, a importação não salvará o registro do console sem o dado do fabricante do mesmo. Entretanto, irá gerar um log de erro que você poderá verificar posteriormente o que não foi salvo.
