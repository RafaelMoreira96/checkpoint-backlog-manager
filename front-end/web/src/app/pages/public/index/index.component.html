Sobre o projeto

Originalmente conhecido como "Game Beating Project", o CheckPOINT é um sistema simples, desenvolvido com Go e Angular, que surgiu como um projeto de estudo para a implementação de novas tecnologias.

A versão inicial do projeto era uma tabela em planilha, onde os jogadores registravam os jogos que completavam. Além disso, outras planilhas derivadas forneciam dados valiosos, como estatísticas de jogos zerados por gênero, plataforma e ano de lançamento.

Após analisar a eficácia dessa abordagem, decidimos evoluir o projeto, transformando-o no CheckPOINT, uma plataforma mais robusta e eficiente, que visa facilitar o acompanhamento da jornada gamer de maneira intuitiva e organizada.

Na CheckPOINT, você conseguirá, de maneira prática e intuitiva, criar uma lista de jogos que você já zerou. Após efetuar o registro, sua lista de jogos zerados começa a ser criada e você poderá ver a sua jornada gamer cada vez mais aumentando!

Obs.: Essa é a versão beta, ja disponível para o público. Será um projeto em constante atualização, visando a melhoria para os usuários. Ainda temos um planejamento, como:

  • Integração com a API do IGDB para melhorar o processo de registro e importação de CSV.
  • Melhoria no modo escuro.
  • Responsividade para dispositivos móveis.
