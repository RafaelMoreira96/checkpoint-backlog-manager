Modo CSV

Inserir lista de gêneros

Insira um arquivo CSV com os gêneros que deseja adicionar ao banco de dados.
O arquivo deve conter uma coluna com o nome do gênero apenas.

Inserir lista de fabricantes

Insira um arquivo CSV com os fabricantes que deseja adicionar ao banco de dados.
O arquivo deve conter uma coluna com o nome do fabricante e ano de fundação.

Inserir lista de consoles

Insira um arquivo CSV com os consoles que deseja adicionar ao banco de dados.
O arquivo deve conter uma coluna com o nome do console, a fabricante e ano de lançamento do console.
